## How to Use:

1. **Open Chatbot**: Click the green chat button (bottom-right corner)
2. **Type Question**: Ask anything in natural language
3. **Get Answer**: Instant response with helpful info
4. **Quick Replies**: Click suggested buttons for common questions
5. **Close**: Click X or chat button again

## Technical Details:

- **File**: `chatbot.js` - Main logic
- **Styles**: `chatbot.css` - UI design
- **Knowledge Base**: Pre-programmed responses for 50+ topics
- **No Backend Required**: Runs entirely in browser
- **Lightweight**: ~15KB total size

## Customization:

The chatbot can be easily extended by adding more keywords and responses in the `knowledgeBase` object in `chatbot.js`.
